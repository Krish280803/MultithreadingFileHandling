# Multithreading File Handling

This is a simple Java project based on multithreading and file handling.

The program reads multiple text files using threads and stores the results in a log file.

 Concepts Used
- Thread class
- BufferedReader
- BufferedWriter
- File handling
- Multithreading

 Tools Used
- Java
- Eclipse IDE
